HTML

  • Enter your HTML notes here
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The metadata is not seen by users
  • The section elements contains thematic content
  • You can create lists within de body elements as unordered lists
  • All of the elements must be well defined and ordered
  • Elements are nested inside other elements are called child elements
  • The element that holds all the children is known as parent element
  • You can create breaks or spaces between elements

Css

  • Language use by developers to define how webpage content is presented to users.
  • There are three ways to style a webpage using CSS: inline, internal and external
  • Declarations contain two important components: th css property and the value of the property
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git pull: Fetches and integrates changes from a remote repository.
  • git push: Pushes local changes to a remote repository.
  • git checkout : Switches to a different branch.
  • git commit -m "message": Commits staged changes with a descriptive message.
